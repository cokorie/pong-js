# PONG

### __Technologies Used__
##### _HTML_
  The HTML code is used to house the: 
  * Game title
  * Font style
  * Game window
  * Game score
  * Game reset button

###### _CSS_ 
  The CSS code is used to house the styling of the:
  * Game window sizing
  * Title font
  * Scoreboard styling
  * Reset button styling

###### _JavaScript_ 
  The JaveScript code is used to house the functions & logic of:
  * Rendering of the game board, scores and reset button
  * Player paddle sizing, speed and  colors
  * Game ball sizing, speed, direction and color
  * Starting and resetting the game
  * Checks for ball impact on paddles and player side for scoring
